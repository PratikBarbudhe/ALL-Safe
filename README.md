
  # Cybersecurity Dashboard Design

  This is a code bundle for Cybersecurity Dashboard Design. The original project is available at https://www.figma.com/design/Sbn1stA1AdoVhC9RYTNk4A/Cybersecurity-Dashboard-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Desktop (Tauri)

  AllSafe ships as a Windows desktop app via [Tauri 2](https://v2.tauri.app/). See [docs/TAURI_SETUP.md](docs/TAURI_SETUP.md) for prerequisites, `npm run tauri:dev`, and `npm run tauri:build`.

  ## Python backend

  Local FastAPI service for system monitoring. See [backend/README.md](backend/README.md) for setup, [docs/DASHBOARD.md](docs/DASHBOARD.md) for the live dashboard, and [docs/PROCESS_MONITOR.md](docs/PROCESS_MONITOR.md) for process monitoring.
